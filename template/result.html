More Information

This model uses a Passive Aggressive Classifier with the parameters of max_iter=50, C=0.5. The dataset has 20,800 different articles with 10413 “Fake” and 10387 “Real”. The test size was 25% and the accuracy was 96.43%. Additionally, the K fold accuracy was 96.28% when cv=5. To test the model further, testing was done with a different dataset. Using this dataset, the accuracy was 76.43%. Limitations: The datasets used were from 2016-2018 based on US politics hence the model works best with US political articles. See the code on github. This website was made using Flask as a backend and HTML, CSS and Java Script for frontend and validation.
